# 100만원 모으기 💰

절약하고 모은 돈을 기록하며 목표를 채워가는 저금 챌린지 웹앱.

## 사연
이제 막 취업한 직장인이 동료의 휴대폰을 고장 내 **96만원**을 물어주게 된 날,
"100만원을 모을 때까지 쓰겠다"고 다짐하며 만든 앱.

## 기능
- 💸 절약 / 🐷 저축 기록, 저금통 채우기 시각화, 진행률
- 🏆 목표 달성 시 수상소감 작성 → **명예의 전당**에 박제
- 🚀 새 목표 선택 (10만 ~ 1000만원 / 직접 입력), 🌴 잠깐 쉬기
- 💾 데이터는 브라우저(localStorage)에 저장 — 사용자마다 독립
- 📱 PWA: 폰 홈 화면에 추가하면 앱처럼 실행

## 로컬에서 실행
```bash
node server.js
# http://localhost:8753 접속
```
